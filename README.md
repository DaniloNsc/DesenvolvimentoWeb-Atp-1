## 💡 Sobre o Projeto

Na **semana 4** do curso, foi apresentado o conteúdo de **arrays em React**, encerrando o módulo de criação de um único componente. Com base nos conteúdos das 4 primeiras semanas, foi desenvolvida esta aplicação de login com as seguintes regras:

- Verificar se o e-mail digitado é igual a `eduardo.lino@pucpr.br`
- Verificar se a senha digitada é igual a `123456`
- Exibir **"Acessado com sucesso!"** se as credenciais forem corretas
- Exibir **"Usuário ou senha incorretos!"** se algum campo estiver errado
